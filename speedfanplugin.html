SpeedFan Plugin for LCD Smartie

Download

SpeedFan Plugin Alpha 1 can be downloaded here.

Install

  • To install this plugin copy the speedfan.dll into the plugins directory of LCD Smartie. [Only the 5.3 final and above builds are supported.]

Using the plugin

To use the plugin you will need to insert commands into the your screens.
The command is $dll(speedfan,[func],[num],0). Where [func] is 1 for fan speeds, 2 for temp sensors, 3 for volt sensors. [num] is the sensor number. So, $dll(speedfan,2,3,0) will display the 3rd temp sensors reading.
