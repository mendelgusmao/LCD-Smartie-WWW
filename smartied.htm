Latest beta release

LCD Smartie has been open source since 29-Oct-2004. Since then work has been carried out to produce the first open-source release. Our main aim is to ensure all features work and that it is robust and stable.

The latest beta release is 5.3 beta 3. [Release Notes]

Plugins

Some of these plugins will only work on Smartie 5.3 and above.

Official plugins - made by the Smartie team

  • Perf Plugin: bar-graphing of cpu-load (and others) over time.
  • Big Num Plugin: Display large (2 or 4 line high) numbers.
  • Menu Plugin: Create your own menus.
  • WinAmp spectrum analyzer: Display spectrum analyzer graphs!

Other plugins

  • Sage TV Plugin by Anders Nolberger. Plugin and source provided!
  • Meedio Plugin by Harald Wagner. The Smartie plugin is part of his add-in for Meedio.
  • iTunes Plugin by Gareth Price
  • (coming soon - being tested) Beyond TV Plugin by Kevin Worrel
  • (coming soon) Windows Media Center Plugin by FrontView.
  • (coming soon) Image/bar plugin.
  • (coming soon) clock plugin.

Config files/Screen ideas

LCD Smartie comes with a config file that includes a few demo screens - but many of these will need adjusting for your screen size. Many plugins also come with their own demo config file with demo screens.
