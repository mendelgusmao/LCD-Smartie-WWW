Latest offical release

The latest offical release is 5.2 (dated 13-4-2003). This was the last build that BasieP (the orginal author) created. You can download this release but you may find the latest alpha release of 5.3 to be more robust and stable.

Latest alpha release

LCD Smartie has been open source since 29-10-2004. Since then work has been carried out to produce the first open-source release. Our main aim to be ensure all features work and that it is robust and stable.

The latest alpha release is 5.3 alpha 5. [Release Notes]
